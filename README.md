# Détermination de l’origine de réplication
Projet de bio-informatique portant sur l’analyse de génomes afin
d’identifier l’origine de réplication.

## Objectifs
- Calcul du taux GC
- Calcul du GC skew par fenêtre glissante
- Analyse comparative de séquences génomiques

## Technologies utilisées
- Python
- R

## Contenu du dépôt
- Scripts Python pour l’analyse des séquences
- Scripts R pour la visualisation
- Fichiers de séquences génomiques
- Rapport PDF expliquant la démarche et les résultats
